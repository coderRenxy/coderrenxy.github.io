作息搭配

前言 每个人都有适合自己的作息，不用随大流，找到自己的生活、学习节奏就好。 第一更 我的一天 第一次找到不用依靠外界环境逼迫性学习的时间规律是：23点-次日5点学习，一觉睡到13点，躺着刷会手机就15点，然后慵懒的起床吃饭，这里有个习惯就是，一定是看剧下饭！ 白天总是不想学习，在晚上昏暗、安静、手机不可能弹出会话框的时候是最适合我的学习时间。 再之后就是回到学校，不能再阴间作息于是逼迫自己白天学习，晚上摆烂+睡觉，出现很严重的效率问题：静不下来。 对我来说，白天很难学下去，白天的学习只能有外界压迫。但是最近发现一件事：刚起床那会很容易进状态，只要起床不看手机不刷快餐短视频，一天的节奏还没有被加速。 所以我现在每天起床，什么都不管不干，只学习，学到吃午饭就是分水岭了，午饭一吃，心思全无，状态低迷，什么都没有心思一样，等天暗下去慢慢就好了。 ...
